# Site Psicóloga Maria do Bairro

Site profissional para psicóloga com versões em Português e Inglês.

## 🚀 Como Publicar no Vercel

### 1. **Crie uma conta no GitHub**
- Acesse https://github.com
- Clique em "Sign up" e crie sua conta gratuita

### 2. **Crie um novo repositório**
- Clique em "+" → "New repository"
- Nome: `site-psicologa-maria`
- Marque "Public"
- Clique "Create repository"

### 3. **Faça upload dos arquivos**
- Na página do repositório, clique em "Add file" → "Upload files"
- Arraste todos os 4 arquivos:
  - `index.html`
  - `en.html` 
  - `style.css`
  - `script.js`
- Clique "Commit changes"

### 4. **Crie conta no Vercel**
- Acesse https://vercel.com
- Clique "Sign up" e faça login com sua conta GitHub

### 5. **Deploy automático**
- No Vercel, clique "New Project"
- Selecione seu repositório `site-psicologa-maria`
- Clique "Deploy"

### ✅ Pronto!
Seu site estará disponível em:
`https://site-psicologa-maria.vercel.app`

## 📧 Personalização

### Para personalizar:
1. Edite os arquivos diretamente no GitHub
2. Ou faça download, edite localmente e faça upload novamente

### O que personalizar:
- `index.html`: Dados em português
- `en.html`: Dados em inglês  
- `style.css`: Cores e estilos
- Links e informações de contato

## 🌐 Domínio Personalizado (Opcional)

Para usar um domínio próprio (ex: mariapsicologa.com):
1. Compre o domínio em qualquer registrador
2. No Vercel, vá em "Domains"
3. Adicione seu domínio
4. Configure o DNS conforme instruções
